# APU Go Green: Sustainable Campus Transportation System

**APU Go Green** is a modern, web-based platform designed for the university community to promote eco-friendly commuting. It incentivizes students and staff to reduce their carbon footprint by combining trip tracking, carpooling, and gamified rewards.

###  Key Features
* **Eco-Trip Logging:** Users log daily commutes (walking, cycling, or public transport) to calculate personal CO2 savings and financial impact.
* **Integrated Carpooling:** A marketplace where verified student/staff drivers offer rides. Features include seat booking, driver verification workflows, and live broadcast alerts.
* **Digital Wallet & Payments:** An integrated payment system for seamless ride transactions, including a mock payment gateway for balance top-ups.
* **Gamification & Rewards:** Users earn "Green Points" for every sustainable action, redeemable for campus perks like cafe vouchers.
* **Admin & Analytics:** A central dashboard for administrators to verify drivers and view real-time environmental impact analytics.

###  Technical Stack
* **Backend:** PHP (Session-based authentication, prepared SQL statements)
* **Database:** MySQL (Relational schema with automated table creation)
* **Frontend:** Responsive UI built with **Tailwind CSS**, Lucide Icons, and JavaScript.

###  Quick Start
1. Create a database named `apu_sustainable_transport`.
2. Run `index.php` (Tables will auto-generate on the first load).
3. Use the **"Initialize Demo Data"** button on the login page to set up test accounts.
